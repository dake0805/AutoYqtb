# AutoYqtbSubmit

## 自动疫情填报 西北工业大学

http://yqtb.nwpu.edu.cn/

## 使用

https://yqtb.xqjun.top/

## 环境要求

python 3

mysql (optional)

jdk 8

## 功能

- 翱翔门户 CAS登录

  `login.py`

- 疫情填报

  `yqtb.py` `main.py`

    - 在学校
    - 指定城市

- 基于Spring的用户信息收集及管理

    - 可用于多人统一批量填报时，收集及存储用户资料用。

    - 登记后可查看及修改已注册的信息

- 配合 Linux 的定时任务，可实现每日自动填报。
